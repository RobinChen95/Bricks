+ 实现的功能:
1. 五个关卡以及不同硬度的砖块</br>
2. 消灭砖块掉下道具的功能</br>
3. 球拍的长短变换的道具、增加生命的道具、增加球速的道具等</br>
4. 没有调用任何第三方API</p>
